## 순서없는 목록 만드는 태그
<ul>
 <li>항목1</li>
 <li>항목2</li>
</ul>
 
## 간단한 표만들기 실습
<table>
 <tr>
  <td>1</td>
  <td>2</td>
 </tr>
 <tr>
  <td>3</td>
  <td>4</td>
 </tr>
</table>

요약이 있는 부분으로 표의 구조를 나눌때 <thead>,<tbody>,<tfoot>를 이용한다.
 
## 이미지 삽입 예
``` <img src="이미지 파일 주소" alt="텍스트"> ```
 
## 오디오와 비디오 삽입 예
  오디오 ``` <audio scr="오디오 파일 경로"></audio> ```
  비디오 ``` <video src="비디오 파일 경로"></video> ```
  
## 하이퍼링크 삽입 예
  ``` <a herf="링크할 주소">텍스트,이미지</a> ```
