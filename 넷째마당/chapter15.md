## 함수
### 함수 선언
+ function 함수명() { 명령 }

### 함수 호출
+ 함수명() 또는 함수명(변수)
 
### var를 사용한 변수의 특징
+ 스코프 : 변수를 선언하고 사용할 때 변수가 적용되는 범위
+ 지역 변수 : 함수 안에서만 쓸 수 있는 지역 변수
+ 전역 변수 : 스크립트 안에서 자유롭게 쓸 수 있는 전역 변수

### let을 사용한 변수의 특징
+ let 예약어로 선언한 변수는 변수를 선언한 블록 ({}로 묶은 부분)에서만 유효하고 블록을 벗어나면 사용할 수 없음
+ var 예약어를 사용한 변수는 선언하기 전에 실행하더라도 아직 할당되지 않은 자료형인 undefined값을 가질 수 있음 (호이스팅) let 예약어를 사용한 변수는 선언하기 전에 사용할 경우 오류 메시지를 나타냅니다.

### const를 사용한 변수의 특징
+ const 예약어는 프로그램 안에서 변하지 않는 값인 상수 이러한 값을 선언할 때 사용

### 자바스크립트 변수 사용법
+ 전역 변수 사용 최소화
+ var 변수는 함수의 시작 부분에서 선언
+ for문에서 카운터 변수를 사용할 때는 var예약어를 사용하지 않음
+ ES6을 사용하면 예약어var보다 let을 사용하는 것이 좋음

### 즉시 실행 함수
+ (function() { 명령 }()):
+ (function(매개변수) { 명령 }(인수));
+ 함수를 정의와 동시에 실행

### 화살표 함수
+ (매개변수) => { 함수 내용 }
+ 함수 선언을 좀 더 간단하게 작성, 익명 함수에서만 사용
